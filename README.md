# 后端开发模板

### 主流框架
- SpringBoot 2.7.X
- Spring AOP 切面编程
- Spring Schedule 定时任务
- Spring 事务注解

### 数据存储
- mysql
- redis
- elasticsearch
- 对象存储

### 工具类
- EasyExcel excel表格处理
- Hutool 工具类
- Apache Commons Lang3 工具类
- Lombok 注解
- Swagger 接口文档

- Spring Session Redis 分布式登录
- 全局请求，响应，异常拦截器
- Swagger + Knife4j 接口文档
- 自定权限注解 + 全局校验
- 全局跨域处理
- 精度丢失
- 多环境配置
